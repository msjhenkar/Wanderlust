# 🌍 WanderLust

A travel platform where users can explore the properties and book them for their stay and hosts can manage their properties.

## 📂 Project Structure 
Wanderlust/ 
│ 
├── backend/ # Django REST API 
└── frontend/ # React App 

## 💡Features
-🔐 Authentication (JWT-based login/signup)
-👤 Role-based access (User / Host)
-🏠 Property listings
-📊 User dashboard
-🌐 Full-stack integration (React + Django)

## Tech Stack
#### Frontend
    React
    Axios
    
#### Backend
    Django
    Django REST Framework
    PostgreSQL
    JWT Aunthetication
